------
Как добавить ученика:
Откройте файл /course/index.html в редакторе кода. Найдите объект STUDENTS в самом начале файла и добавьте туда новую строку с кодом и датой старта потока: 
// Пример добавления:
const STUDENTS = {
    'DEMO01': { ... },
    'НОВЫЙ_КОД': { name: 'Имя ученика', start: 'YYYY-MM-DD', course: 'recovery', days: 14 },
};
