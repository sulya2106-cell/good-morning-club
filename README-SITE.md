Привет, Айса! 👋
Этот документ создан специально для тебя. Здесь собрано всё, что нужно знать, чтобы самостоятельно управлять сайтом, добавлять видео, менять тексты и цены без помощи программистов.

---

## 📁 Из чего состоит сайт (Структура файлов)

* **`content.js`** — **САМЫЙ ВАЖНЫЙ ФАЙЛ**. Здесь хранятся все карточки с видео и текстами для библиотеки. Именно его ты будешь менять чаще всего.
* **`index.html`** — Главная страница (Лендинг).
* **`app.html`** — Закрытая библиотека практик (доступна только после входа).
* **`course-cleanse.html`** — Страница курса "Восстановление тела".
* **`course-fire.html`** — Страница курса "Внутренний огонь".
* **`about.html`** — Страница "О преподавателе".
* **`manifest.json`** и картинки (`icon-192.png`, и т.д.) — Настройки для того, чтобы сайт работал как приложение на телефоне.

## ✏️ Как поменять текст на главной (и других) страницах

1. Открой нужный файл, например `index.html`.
2. Я оставил для тебя подсказки: `<!-- Поменяй текст ниже -->`.
3. Просто найди нужный текст между тегами (например, между `<p>` и `</p>`) и аккуратно замени его на новый. 
4. **Важно:** не удаляй символы `<` и `>`, они нужны для разметки.

## 🎥 Как добавить новое видео в библиотеку

1. Открой файл `content.js`.
2. Найди массив `PRACTICES`. Там есть категории (body, recovery, breath, nauli, theory).
3. Скопируй шаблон нового видео и вставь его в нужную категорию:
```javascript
{
    t: 'Название твоего видео',
    c: 'body', // код категории
    d: '20 мин', // длительность
    lv: 'Все уровни', // уровень
    desc: 'Описание...',
    vk: 'ССЫЛКА_ИЗ_ВК',
    e: '🧘' // эмодзи
},
```
4. **Обязательно** поставь запятую `,` после закрывающей фигурной скобки `}`, если после этого видео идет еще одно!

## 📝 Как добавить новую текстовую карточку (статью/чек-лист)

1. Открой файл `content.js`.
2. Прокрути вниз до массива `TEXTS`.
3. Скопируй и вставь шаблон:
```javascript
{
    t: 'Название на карточке',
    c: 'theory',
    e: '📖',
    sub: 'Короткий текст',
    h1: 'Главный заголовок внутри',
    h2: 'Подзаголовок',
    body: '<p>Твой текст здесь. Для новых абзацев используй теги p.</p>'
},
```

## 🔗 Как получить правильную ссылку на видео из VK

Для поля `vk` нужна специальная ссылка для встраивания, а не обычная из адресной строки!

1. Открой видео ВКонтакте с компьютера.
2. Нажми кнопку **"Поделиться"** (стрелочка) под видео.
3. Выбери вкладку **"Экспортировать"**.
4. В поле "Код для вставки" найди часть, которая начинается с `src="`.
5. Скопируй **ТОЛЬКО** саму ссылку внутри кавычек.
   *(Она должна выглядеть примерно так: `https://vk.com/video_ext.php?oid=-123...` или `https://vkvideo.ru/video_ext.php?...`)*
6. Вставь эту ссылку в поле `vk` в файле `content.js`.

## 💰 Как поменять цены на страницах курсов

1. Открой `course-cleanse.html` или `course-fire.html`.
2. Найди комментарий `<!-- 💰 ЦЕНЫ — меняй цифры ниже -->`.
3. Под ним будет строчка с ценой, например: `<div class="price-val">5 000 ₽</div>`.
4. Замени `5 000` на нужную цифру и сохрани.

## 💬 Как добавить новый отзыв

1. Открой страницу курса (`course-cleanse.html` или `course-fire.html`).
2. Найди комментарий `<!-- 💬 ОТЗЫВЫ — добавь новые по шаблону ниже -->`.
3. Скопируй блок отзыва:
```html
<div class="testi-card">
    <div class="testi-author">Имя Ученика</div>
    <div class="testi-text">"Текст отзыва..."</div>
</div>
```
4. Вставь его перед другими отзывами и поменяй имя и текст.

## 📸 Как добавить/поменять фотографию

1. Загрузи новую фотографию (лучше в формате `.jpg` или `.png` и сжатую, чтобы сайт грузился быстро) в ту же папку, где лежат все файлы сайта.
2. В нужном HTML файле найди место с комментарием `<!-- Замени src="..." на путь к фото -->` (или просто тег `<img>`).
3. В атрибуте `src` напиши название твоего файла. 
   *Пример:* `<img src="my-new-photo.jpg" alt="Описание фото">`

## 🚀 Как сохранить изменения (чтобы они появились на сайте)

Если ты работаешь прямо на GitHub:
1. После того как изменила файл, прокрути страницу наверх или вниз и нажми зеленую кнопку **"Commit changes..."**.
2. В появившемся окне снова нажми зеленую кнопку **"Commit changes"**.
3. Подожди 1-2 минуты. GitHub автоматически обновит сайт, и новые данные появятся по вашей ссылке!

## 🛑 ЧТО НЕЛЬЗЯ ТРОГАТЬ (Опасно!)

* В файле `app.html` в самом низу есть блок: `<!-- ⚠️ НЕ ТРОГАЙ НИЖЕ — JAVASCRIPT ЛОГИКА И FIREBASE ⚠️ -->`. **Не меняй там ничего!** Это настройки базы данных и авторизации, если их сломать — ученики не смогут войти в библиотеку.
* Не удаляй случайно символы разметки в HTML (вроде `<div>`, `</section>`).
* В файле `content.js` не забывай про кавычки `' '` вокруг текста и запятые `,` между элементами.

## 🩹 Частые ошибки и как их исправить

* **Сайт перестал грузиться белый экран (в библиотеке):** 
  Скорее всего, в файле `content.js` пропущена запятая `,` или случайно удалена кавычка `'`. Зайди в файл и проверь последние изменения.
* **Текст поехал или выглядит странно:**
  Случайно удалилась угловая скобка `<` или `>` в HTML. Проверь теги вокруг твоего текста.
* **Видео не работает:**
  Проверь ссылку `vk` в `content.js`. Убедись, что скопирована именно ссылка для экспорта (`video_ext.php...`), а не обычная ссылка на пост.
* **Внесены изменения, но на сайте старая версия:**
  Обнови страницу с очисткой кэша (на компьютере нажми `Ctrl + F5` или `Cmd + Shift + R` на Mac). Иногда браузер запоминает старую версию.
