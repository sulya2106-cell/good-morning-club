## Как добавить видео

1. Зайди на GitHub → открой файл `content.js` → нажми ✏️ (карандашик)
2. Найди раздел нужной категории (например `// ── Телесные практики ──`)
3. Скопируй любой блок `{ ... },` и вставь после него
4. Заполни поля:

```javascript
{
    t: 'Название практики',         // заголовок карточки
    c: 'body',                      // категория (см. ниже)
    d: '30 мин',                    // длительность
    lv: 'Все уровни',              // уровень: Все уровни / Средний / Подготовленные
    desc: 'Описание практики.',     // текст под видео
    vk: 'ССЫЛКА_ИЗ_VK',           // VK embed (см. ниже)
    e: '🧘'                        // эмодзи-заглушка (видно пока нет видео)
},
```

**Категории:**
| Код | Раздел |
|-----|--------|
| `body` | 🧘 Телесные практики |
| `recovery` | 💆 Восстановление тела |
| `breath` | 💨 Дыхательные практики |
| `nauli` | 🌀 Работа с наули |
| `theory` | 📖 Теория |

**Как получить VK embed ссылку:**
1. Открой видео в VK
2. Нажми "Поделиться" → "Экспортировать" (или "Код для встраивания")
3. Скопируй то что внутри `src="..."` — это будет типа:
   `https://vkvideo.ru/video_ext.php?oid=-12345&id=456789&hash=abc123&hd=3`
4. Вставь эту ссылку в поле `vk:`

5. Нажми **Commit changes** внизу страницы
6. Сайт обновится через 1-2 минуты

**⚠️ Запятая в конце блока обязательна!** Если забудешь — сайт сломается.

## Как добавить текстовую карточку

В файле `content.js` найди массив `TEXTS` и добавь блок:

```javascript
{
    t: 'Название на карточке',       // что видно в ленте
    c: 'body',                       // категория
    e: '📋',                         // эмодзи на карточке
    sub: 'Подзаголовок',            // серый текст под названием
    h1: 'Большой заголовок',         // в попапе, крупный
    h2: 'Подзаголовок попапа',       // в попапе, серый
    body: '<p>Текст...</p>'          // HTML-контент попапа
},
```

**HTML-теги для body:**
- `<p>Абзац текста</p>` — обычный текст
- `<strong>Жирный</strong>` — выделение (будет зелёным)
- `<ul><li>Пункт</li></ul>` — список
- `<a href="URL" target="_blank">Ссылка</a>` — ссылка
- `<br>` — перенос строки

## Как одобрить нового ученика

1. Ты получишь уведомление в Telegram: "🆕 Новая регистрация"
2. Зайди на сайт → войди как admin → нажми 👑 **Админ**
3. Найди ученика в таблице (будет с ❌ в статусе)
4. Выбери нужные категории чекбоксами **ИЛИ** нажми пресет:
   - **Тело** = Телесные + Дыхательные + Теория
   - **Очищение** = Восстановление + Дыхательные + Теория
   - **Наули** = Наули + Дыхательные + Теория
   - **Полный** = Всё
5. Статус автоматически станет ✅
6. Ученик обновит страницу и увидит свой контент

## Как заблокировать ученика

1. Админка → найди ученика → нажми **Блок**
2. При следующем входе увидит "Доступ заблокирован"

Чтобы разблокировать — нажми **Разблок**

---

## Как сбросить привязку устройства

Если ученик сменил телефон и не может войти:

1. Админка → найди ученика → нажми **Сброс**
2. Привязка обнулится, ученик сможет войти с нового устройства

По умолчанию разрешено 2 устройства на аккаунт.

---

## Как удалить видео или карточку

1. Открой `content.js` на GitHub → ✏️
2. Удали весь блок `{ ... },` нужной карточки
3. Commit changes

---

## Если сайт сломался

Скорее всего забыта запятая или кавычка в content.js. Как починить:

1. Зайди на GitHub → `content.js` → **History** (вверху справа)
2. Найди последний рабочий коммит → нажми на него
3. Нажми `...` → **Revert changes** (или скопируй старый код)

GitHub хранит всю историю — ничего не теряется.

### Firestore Rules (ВАЖНО!)
Зайди в Firebase Console → Firestore → Rules → замени на:

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId} {
      allow read: if request.auth != null && request.auth.uid == userId;
      allow write: if request.auth != null && request.auth.token.email in ['sulya2106@gmail.com'];
    }
  }
}
```
